8. Critical Design Rules
✓ SAFE PAIRS
White / Warm White on any dark bg
Gold (#FBBF24) on #110826 or #231550
Green (#22C55E) on any dark bg
Orange (#FF6B00) on any dark bg
Cyan (#22D3EE) on any dark bg
Dark text (#2D1B69) on any survey light bg
White on purple button (#A855F7)
✗ AVOID
Purple text on card bg — 3:1 FAIL for small text
Muted (#9B7FD4) on sidebar for <14px text
Any light text on light survey backgrounds
Any dark text on dark game backgrounds
Transparent bg on any element — always explicit
body without background color declared
🏗 ARCHITECTURE RULES
body always: background #110826
Survey: position:fixed, z-index:50, bg #FAF8FF
All game CSS variables in :root only
Survey CSS scoped under #surveyScreen
Fredoka One = numbers / scores / headings
Nunito = everything else
main-c always: background #1A0F3A explicit
