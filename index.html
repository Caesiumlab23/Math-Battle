10. Design Rules — v2
✓ SAFE PAIRS
White / EDE9FF on any dark purple bg
Gold (#FBBF24) on #1A0533 or #2D1B69
Green (#22C55E) on any dark bg — AAA
Orange (#FF6B00) on #1A0533 — AA
Blue (#38BDF8) on any dark bg — AAA
Dark text (#1A0800) on gold button
Dark text (#0c1a2e) on blue button
✗ NEVER DO
Purple (#A855F7) as small body text on cards
Muted (#A78FD4) on card bg at <14px
Red (#FF4444) small text on cards
White text on gold / yellow buttons
Fredoka One in italic style
Nunito weight 400 — use 700 minimum
ℹ️ ARCHITECTURE
body always: background #1A0533
Survey: position:fixed, z-index:50, bg #F5F7FF
All game CSS variables in :root only
Survey CSS scoped under #surveyScreen
Fredoka One = numbers / scores / questions
Nunito weight 700–900 = everything else
Glows: box-shadow rgba with 40–50% opacity
