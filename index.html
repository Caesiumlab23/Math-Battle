8. Critical Design Rules
✓ SAFE PAIRS
White / E2E8F8 on any dark BG
Gold (#F59E0B) on #080B1A or #161C3A
Green (#10B981) on any dark BG
Orange (#F97316) on any dark BG
Dark text (#1E1B4B) on any survey light bg
White on purple button (#7C3AED)
✗ AVOID
Purple (#7C3AED) text on card BG — 3.0:1 FAIL
Muted (#7B88B8) on sidebar for <16px text
Red (#EF4444) on cards for small body text
Light text on light survey backgrounds
Dark text on dark game backgrounds
ℹ️ ARCHITECTURE RULES
body always: background #080B1A
Survey: position:fixed, z-index:50, bg #F5F7FF
All game CSS variables in :root only
Survey CSS scoped under #surveyScreen
Fredoka One = numbers / scores / headings
Nunito = everything else
